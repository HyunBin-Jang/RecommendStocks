# RecommendStocks
데이터베이스설계 기말프로젝트 - 포트폴리오 유사도 기반 주식 추천

## 프로젝트 소개
저명투자자과 사용자의 주식 포트폴리오를 비교하여 유사도를 측정하고 이에 따라 사용자에게 주식 종목을 추천

## 개발 환경
Java 17<br/>
SpringBoot 3.3.5<br/>
Mysql 8.0<br/>
Flask<br/>
